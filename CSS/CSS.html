Linking a stylesheet

When you start a CSS document you do not have to declare the document type or put any special code to start the document however in the html document you need to provide a link to the stylesheet like so:

<link rel="stylesheet" type="text/css" href="CSS.css">

Colours in CSS

There are four ways of defining a clouour in CSS The first way which you have already come accross is using named colours of which there are 147 the second is using RGB values rangeing from between 0 and 255 the third is to use a hexadicimal color codes (more on those later) and the last way is using HSL colours.

Here is an example of setting colour values using RGB

p {
color: rgb(255, 0, 0);
background-color: rgb(61,255,255);
}

Hexadecimal colour codes are another way to specify colour as they offer a huge range of colours (16,777,216) to be exact however unlike rgb colours that are specified in base 10 hexadecimal colours are written using base 16 so for example #000000 is black and #ffffff is white.

An advantage of using this colour scheme is that only two characters at most are ever needed per colour and the order of the colours is the same as the rgb scheme (the # at the start identifies it as hex code colour) it also means that if the colour is made of the same characters it can be abbreviated like so #ff0

here is an example of setting colour values using hexidecimal code

p { color: #3d3dff; background-color: #ff0; }

HSL stands for Hue Saturation and Lightness
Hue is What we understand as colour and is represened on what is known as a color wheel and can take any value betweeen 0 and 360
Saturation is the amount of gray in a colour and is a percentage between 0% and 100% with 0% being gray and 100% being full saturation (no gray)
lightness is the amount of white in a colour it is a percentage with 0% being black 50% is normal and 100% is white

p { color: hsl(182, 20%, 50%); background-color: hsl(300, 100%, 40%); }

Font Formatting

Font Family

You can change the font family using the 'font-family' property as in the example below

h1 { font-family: Garamond; }

Font Size

Font size is measured in three units

The First Unit is Pixels(px)

p { font-size: 32px; }

The Second Unit is Ems ems are a relative unite of measurement as it changes the size of the text relative to the size of the elaments size of text

p { font-size: 3em; }

The Third and final unit of font size is % which is also a relative unit of measurement relative the the default text size of 16px so for example if the size was set to 200% the size would be equivelent to setting it to 32 px

p { font-size: 200%; }

Line Height

The line-height property can also be set in either pixels or ems ems being the more preferable as it is relative to size of the text on the page.
