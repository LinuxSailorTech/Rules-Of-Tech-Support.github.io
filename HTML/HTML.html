Hyper Text Markup Language (HTML)

HTML is used as one of the main building blocks of the web along with CSS and JavaScript

When you start a HTML document the first thing you do is declare the type of document (officially known as a document type declaration) using:

<!DOCTYPE html>

Once you have declared the document type you are then ready to start coding in HTML. The HTML code itself is contained within the tags:

<html>

</html>

All the code between the angle brackets < > is known as a html elament most html elaments have an openging <p> and a closing </p> tag however some elaments such as the <link> tag are self closing

The head elament contains information releating to the webpage and is used by the browser.
it contains information such as the title of the page a link to the stylesheet which is written in css it may also contain a link to a font bank.
here is an example header:

<html>

<head>

<link href='https://fonts.googleapis.com/css?faimily=Roboto:200'>

<link rel="stylesheet" type="text/css" href="CSS1.css">
